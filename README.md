# SleepEZ
### A Spigot Plugin

SleepEZ is a simple sleep plugin that lets you enter a percentage of people that need to sleep to skip night.
If you don't want certain players to be counted against the total number of online players when someone sleeps,
give them the sleepez.bypass permission.

This is my very first plugin! Any feedback is welcome!
```
permissions:
  sleepez.bypass:
    description: Allows user to not be counted toward total player count for sleep events.
    default: false
````
